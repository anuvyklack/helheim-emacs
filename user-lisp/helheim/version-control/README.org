* helheim-diff-hl

Related keys are under =SPC v= prefix in ~diff-hl-command-map~ keymap.

- =[[= / =]]= :: *Prev* / *Next* hunk
- =[{= / =]}= :: View *prev* / *next* hunk
- =[v= / =]v= :: View *prev* / *next* hunk
- =SPC v v= :: View hunk
- =SPC v s= :: Stage hunk
- =SPC v r= :: Revert hunk
- ~SPC v =~ :: Goto hunk
