* Helheim Emacs

#+begin_quote
This is the realm of Hel
#+end_quote

Helheim is Emacs distributing build around [[https://github.com/anuvyklack/hel][Hel]] package. The original idea behind
it was a s

#+begin_quote
*Adwice:* Bind =Caps Lock= to =Esc=, and =Space= to double-action: =Space= on tap and =Ctrl=
on hold.
#+end_quote

** Installation

#+begin_quote
GNU Emacs 29.1 or later is required
#+end_quote

1. Donwload and install "Symbols Nerd Font" from [[https://www.nerdfonts.com/][www.nerdfonts.com]]. This font
   contains only nerd font icons.

2. Clone this repository.

   You can clone it into =~/.config/emacs= so Emacs loads it automatically on
   startup:
   #+begin_src sh
   git clone https://github.com/anuvyklack/emacs-helheim.git ~/.config/emacs
   #+end_src

   Or clone it to any other location and pass the path to Emacs explicitly:
   #+begin_src sh
   emacs --init-directory <path/to/emacs-helheim>
   #+end_src

I decided not to reinvent the wheel, Emacs Lisp already has packages all Helheim
modules are just Emacs packages that you can ~require~ any one you want. You are
in full controll.
